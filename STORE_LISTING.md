WHAT GETS OPTIMIZED?
--------------------------------------------------------------------------------

BEFORE:
"Could you please help me? I was wondering if you could provide a detailed explanation of how machine learning works. I would really appreciate it if you could include some examples."

AFTER:
"Explain machine learning with examples."

What we optimize:
- Removes politeness fluff (please, could you, would you mind)
- Eliminates filler words (just, really, very, basically)
- Condenses verbose phrases (in order to -> to)
- Optimizes structure (provide a detailed explanation -> explain)
- And much more!
